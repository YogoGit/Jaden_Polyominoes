Key Function
any key Resume if paused or start if not started.
escape Pause if not paused.
space bar Drop and lock (hard drop).
down arrow Step down (soft drop).
left arrow Step left.
right arrow Step right.
up arrow Rotate clockwise 90 degrees.
lowercase z Rotate counterclockwise 90 degrees.
lowercase h Puts the active piece on hold;
If a piece was hold, then swaps current piece with on hold piece.
Event Points
soft drop 1 × level number
hard drop 2 × level number × distance dropped
clear 1 row 100
clear 2 rows 300
clear 3 rows 500
clear 4 rows 800
clear 5 rows 1100
clear 6 rows 1300
